# SQL Student Database Project

## 📌 Project Overview

This project demonstrates the creation of a **Student Database using SQL**.
It includes table creation, inserting data, and executing multiple SQL queries.

* Students
* Courses
* Subjects
* Enrollments
* Marks

* INSERT INTO
* SELECT Queries
* WHERE Clause
* LIKE Operator
* ORDER BY
* INNER JOIN

This project contains **24 SQL queries** including:

* Basic data retrieval
* Filtering records
* Sorting data
* Joining multiple tables
